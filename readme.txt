== Description ==

Metrilo is simple, yet extremely powerful Analytics tool. This plugin provides one-click integration with Metrilo's Analytics tracking API for WordPress online stores running on WooCommerce. 


== Installation ==

Once you install the plugin, follow these steps:

1. Activate the plugin.
2. Go to plugin's Settings page
3. Enter your unique Metrilo API token and click "Save settings"

== Frequently Asked Questions ==

= Will it slow down my site? =

Absolutely no. Metrilo's tracking library loads async and doesn't slow down your website at all.

= How long should I wait before I have enough data to trust and act on? =

It really depends on how many visitors do you have per day. It's a good idea to wait at least a week.
